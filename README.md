# Obtained with Ostentation
Warhammer 40,000 themed wallpapers. Mainly in 1080p 16:9. Intended for use with the Darktide mod [Improved Loading Screens](https://www.nexusmods.com/warhammer40kdarktide/mods/1012) (ILS); most of these are from my [Custom Loading Backgrounds](github.com/Backup158/DarktideCustomLoadingBackgroundsList/tree/main) (CLB) list.

From what I could tell, CLS loads everything at the start, while ILS only loads one thing before a loading screen. That makes it not swap mid-loading screen, but should be more performance friendly. >700 wallpapers at startup, even with parallel processing, is a lot.
